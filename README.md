# Prueba Front Ionic

## Pasos para iniciar el proyecto

- Clonar el proyecto (git clone https://github.com/SergioAlberca/prueba-front-sanitas.git).
- Ejecutar 'npm install' en el directoio del proyecto.
- Ejecutar el comando 'ionic serve' para inicializar el proyecto en el directoio correspondiente.
- Ejecutar el comando 'npm run test' para ejecutar los tests en el directoio del proyecto.
